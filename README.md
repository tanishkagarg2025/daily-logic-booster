# Daily Logic Booster

Daily Logic Booster is a Python-based desktop application designed to enhance logical reasoning, problem-solving, and critical thinking skills through interactive challenges and AI-generated questions.

## Features

### Smart Question Generation
- Generates logic and reasoning questions dynamically.
- Supports AI-assisted content generation.
- Provides a variety of challenge types to keep users engaged.

### Progress Tracking
- Tracks user performance and completion history.
- Maintains progress data across sessions.
- Helps users monitor improvement over time.

### Dashboard Analytics
- Displays performance summaries.
- Provides insights into completed challenges and progress trends.
- Offers an organized view of learning activity.

### Personalized Experience
- Customizable application themes.
- User-friendly settings management.
- Clean and intuitive interface for focused learning.

### Modular Architecture
- Structured using multiple Python modules.
- Separation of concerns for maintainability and scalability.
- Easy to extend with additional question categories and features.

## Technologies Used

- Python
- JSON Data Storage
- Object-Oriented Programming (OOP)
- Modular Software Design
